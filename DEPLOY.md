# 🚀 Деплой РецептБука за 15 минут
# Результат: постоянная ссылка вида https://recipebook-xxx.railway.app
# Родственники просто открывают её в браузере — больше ничего не нужно.

═══════════════════════════════════════════════════════
 ШАГ 1 — GitHub (5 минут)
═══════════════════════════════════════════════════════

1. Открой https://github.com и зарегистрируйся (если нет аккаунта)

2. Нажми зелёную кнопку "New" → назови репозиторий "recipebook"
   → нажми "Create repository"

3. Скачай GitHub Desktop: https://desktop.github.com/
   → войди в свой аккаунт
   → File → Clone repository → выбери "recipebook"
   → скопируй все файлы проекта в эту папку
   → нажми "Commit to main" → "Push origin"

   Готово — код на GitHub!


═══════════════════════════════════════════════════════
 ШАГ 2 — Railway (5 минут)
═══════════════════════════════════════════════════════

1. Открой https://railway.app
   → нажми "Login" → "Login with GitHub"
   → разреши доступ

2. Нажми "New Project"
   → выбери "Deploy from GitHub repo"
   → выбери "recipebook"
   → Railway сам определит настройки из railway.toml

3. Добавь базу данных:
   → в проекте нажми "+ New"
   → выбери "Database" → "PostgreSQL"
   → Railway создаст БД и автоматически добавит DATABASE_URL


═══════════════════════════════════════════════════════
 ШАГ 3 — Переменные окружения (3 минуты)
═══════════════════════════════════════════════════════

В Railway → выбери сервис "recipebook" → вкладка "Variables"
→ нажми "Raw Editor" и вставь:

NODE_ENV=production
JWT_SECRET=придумай_любую_длинную_строку_например_бабушкины_блины_2024
JWT_REFRESH_SECRET=ещё_одна_длинная_строка_например_дедушкин_борщ_2024

Нажми "Update Variables" — Railway перезапустит приложение.


═══════════════════════════════════════════════════════
 ШАГ 4 — Создать таблицы в базе данных
═══════════════════════════════════════════════════════

В Railway → выбери PostgreSQL → вкладка "Data"
→ нажми "Query" 
→ скопируй содержимое файла backend/db/schema.sql
→ вставь в поле и нажми "Run Query"

Таблицы созданы!


═══════════════════════════════════════════════════════
 ШАГ 5 — Получить ссылку
═══════════════════════════════════════════════════════

Railway → сервис "recipebook" → вкладка "Settings"
→ раздел "Networking" → нажми "Generate Domain"
→ получишь ссылку вида: https://recipebook-production-abc.railway.app

Эту ссылку отправь всем родственникам!
Можешь добавить её на главный экран телефона как ярлык.


═══════════════════════════════════════════════════════
 ОБНОВЛЕНИЕ ПРИЛОЖЕНИЯ В БУДУЩЕМ
═══════════════════════════════════════════════════════

Когда захочешь что-то изменить:
1. Измени файлы на компьютере
2. В GitHub Desktop: Commit → Push
3. Railway автоматически задеплоит новую версию за ~2 минуты

Родственники обновят страницу — и уже новая версия!

═══════════════════════════════════════════════════════
 АЛЬТЕРНАТИВА: Render.com (тоже бесплатно)
═══════════════════════════════════════════════════════

Если Railway не подойдёт — Render работает аналогично:
1. https://render.com → Login with GitHub
2. New → Web Service → выбери репозиторий
3. Build Command: npm run build
4. Start Command: npm start
5. Add PostgreSQL database
6. Добавь переменные окружения

Минус Render: бесплатный сервис "засыпает" после 15 мин неактивности
(первый запрос после паузы идёт ~30 секунд).
Railway этого не делает.
